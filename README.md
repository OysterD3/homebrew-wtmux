# homebrew-wtmux

Homebrew tap for [wtmux](https://github.com/OysterD3/wtmux) — coordinated git worktrees across sibling repos for AI coding agents.

## Install

```bash
brew install OysterD3/wtmux/wtmux
```

That's it — `brew install <user>/<tap>/<formula>` taps this repo implicitly. No separate `brew tap` step needed.

## Update

```bash
brew update && brew upgrade wtmux
```

## Uninstall

```bash
brew uninstall wtmux
brew untap OysterD3/wtmux
```

## How this tap works

Formulae in `Formula/` are written automatically by [GoReleaser](https://goreleaser.com) on every wtmux release. Source code, issues, and contributions belong on the [main wtmux repo](https://github.com/OysterD3/wtmux), not here.
